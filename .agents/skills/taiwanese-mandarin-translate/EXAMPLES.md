# Taiwan Mandarin translation examples

These examples show the default style this skill should aim for.

## Default vs. formal public-facing register

| Context | Source | Preferred translation |
| --- | --- | --- |
| default Taiwanese Mandarin | Taiwan is a multilingual society. | 台灣是多語言社會。 |
| more neutral public-facing register | Taiwan is a multilingual society. | 臺灣是多語言社會。 |
| default Taiwanese Mandarin | This guide is written for readers in Taiwan. | 這份指南是為台灣讀者撰寫的。 |
| more neutral public-facing register | This guide is written for readers in Taiwan. | 本指南為臺灣讀者撰寫。 |
| default Taiwanese Mandarin | We built this feature for teams in Taiwan. | 我們為台灣團隊打造了這項功能。 |
| more neutral public-facing register | We built this resource for residents in Taiwan. | 本資源供臺灣居民使用。 |

| Source | Preferred translation |
| --- | --- |
| Your changes have been saved successfully. | 你的變更已成功儲存。 |

## Shared Taiwan standard wording

| Source | Preferred translation |
| --- | --- |
| Use wording that is widely understood in Taiwan. | 請使用在台灣普遍容易理解的說法。 |
| This notice is intended for the general public. | 本通知適用於一般大眾。 |
| Keep the language clear and neutral. | 請保持語句清楚、中性。 |

## Rewriting China-leaning wording into Taiwan Mandarin

| Avoid | Prefer |
| --- | --- |
| 請您通過以下鏈接獲取軟件的默認配置信息 | 請透過以下連結取得軟體的預設設定資訊 |
| 服務器返回的數據需要進行一個解析的操作 | 伺服器回傳的資料需要解析 |
| 點擊按鈕後系統會自動加載數據 | 點選按鈕後，系統會自動載入資料 |
| 請先配置接口再激活插件 | 請先設定介面，再啟用外掛 |
| 我們會對接口進行優化以支持更多客戶端 | 我們會最佳化介面，以支援更多用戶端 |
| 通過該功能可以獲取更多視頻信息 | 透過這項功能可以取得更多影片資訊 |

## Literal translation vs. polished Taiwan Mandarin

| Source | Literal or stiff | Better Taiwan Mandarin |
| --- | --- | --- |
| Review the issue before submitting the form. | 在提交表單前審查該問題。 | 送出表單前，請先檢查這個問題。 |
| The app supports offline access. | 此應用支持離線訪問。 | 這個應用程式支援離線存取。 |
| We are working to improve the experience. | 我們正在努力改善體驗。 | 我們正在改善使用體驗。 |
| The deployment failed because the token expired. | 部署失敗，因為令牌過期。 | 部署失敗，因為 token 已到期。 |

## Markdown and placeholders

| Source | Preferred translation |
| --- | --- |
| Click **Copy link** to share this report. | 點選 **複製連結** 即可分享這份報表。 |
| Do not translate `user_id`, `{workspace}`, or https://example.com. | 請勿翻譯 `user_id`、`{workspace}` 或 https://example.com。 |
| Use `--target copilot` to install the skill for GitHub Copilot. | 使用 `--target copilot` 即可為 GitHub Copilot 安裝這個 skill。 |
