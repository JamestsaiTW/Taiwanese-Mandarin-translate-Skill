# Taiwan Mandarin translation examples

These examples show the default style this skill should aim for.

## UI copy

| Source | Preferred translation |
| --- | --- |
| Settings | 設定 |
| Save Changes | 儲存變更 |
| Upload File | 上傳檔案 |
| Download Report | 下載報表 |
| Enable Notifications | 啟用通知 |
| Sign in | 登入 |
| Sign out | 登出 |
| Delete this item? | 要刪除這個項目嗎？ |

## Product and technical writing

| Source | Preferred translation |
| --- | --- |
| Your changes have been saved successfully. | 你的變更已成功儲存。 |
| This feature is enabled by default. | 這項功能預設為啟用。 |
| The server returned an error response. | 伺服器回傳錯誤回應。 |
| Please enter a valid email address. | 請輸入有效的電子郵件地址。 |
| Update the configuration file before restarting the app. | 重新啟動應用程式前，請先更新設定檔。 |
| Click the link below to continue. | 請點選下方連結以繼續。 |

## Rewriting China-leaning wording into Taiwan Mandarin

| Avoid | Prefer |
| --- | --- |
| 請您通過以下鏈接獲取軟件的默認配置信息 | 請透過以下連結取得軟體的預設設定資訊 |
| 服務器返回的數據需要進行一個解析的操作 | 伺服器回傳的資料需要解析 |
| 點擊按鈕後系統會自動加載數據 | 點選按鈕後，系統會自動載入資料 |
| 請先配置接口再激活插件 | 請先設定介面，再啟用外掛 |
