# Taiwanese Mandarin Translate Skill

Translate English into Traditional Chinese with wording that is natural for Taiwan users.

## Files

- `SKILL.md`: Copilot skill definition and translation guidance
- `README.md`: basic usage notes

## Usage

Install this repository as a Copilot skill folder, then use it whenever you need:

- English → Traditional Chinese translation
- Taiwan-preferred wording and terminology
- Preserved formatting for Markdown or structured text
