# English to Traditional Chinese (Taiwan)

Use this skill when the user wants English content translated into Traditional Chinese that reads naturally for Taiwan audiences.

## Translation rules

- Translate into **Traditional Chinese used in Taiwan**.
- Prefer wording commonly used in Taiwan, and avoid Simplified Chinese or China-specific phrasing.
- Preserve the original meaning, tone, and level of formality.
- Keep names, brands, product names, code, commands, file paths, and technical identifiers in the source language unless the user asks otherwise.
- Preserve the original structure and formatting such as headings, lists, tables, and Markdown.
- If the source text is ambiguous and a more accurate Taiwan translation depends on context, ask a brief clarifying question first.

## Terminology preferences

- software → 軟體
- video → 影片
- quality → 品質
- information → 資訊
- internet → 網路
- mobile app → 應用程式
- data → 資料

## Response style

- By default, provide only the translated Traditional Chinese text.
- If needed, make the wording fluent and idiomatic for Taiwan while staying faithful to the source.
